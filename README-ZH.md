# 知网文献综述 API

这是“知网文献综述助手”浏览器插件的后端 API 服务。

浏览器插件负责从知网检索结果页采集论文题录、摘要和关键词；本后端负责用户账号、额度校验、DeepSeek API 调用，以及快速综述和深度综述报告生成。

## 产品定位

本项目不是论文代写工具，也不绕过知网访问限制。它面向正在开题、写论文、做文献综述的硕士、博士等学生，帮助他们在写作前把文献材料整理成可追溯、可下载、可修改的综述素材报告。

核心目标是节省文献综述前期的整理、分类、筛选、对比和归纳时间。

## 使用流程

1. 用户在知网中完成检索。
2. 浏览器插件点击“采集当前页”，形成当前文献集合。
3. 用户登录后，基于题录、摘要和关键词生成快速综述。
4. 快速综述帮助用户筛出值得下载全文的代表论文。
5. 用户通过合法权限获取 PDF，并在插件中导入。
6. 插件提取 PDF 文本并匹配到已采集论文。
7. 后端基于题录、摘要、关键词和 PDF 全文生成深度综述报告。
8. 插件下载报告到用户本地。

## 计划技术栈

- 运行时：Node.js
- 语言：TypeScript
- Web 框架：Fastify
- 数据库：SQLite
- ORM：Prisma
- 部署目标：腾讯云服务器
- 进程管理：MVP 阶段优先使用 PM2

## Python 虚拟环境

如果后续需要 Python 辅助脚本，统一使用项目内 .venv 虚拟环境，不使用系统 Python 环境。当前后端主体仍为 Node.js + TypeScript。

## MVP 后端职责

后端需要提供：

- 用户注册和登录
- 受保护接口鉴权
- 快速综述额度校验与扣减
- 深度综述任务额度校验与扣减
- DeepSeek API 调用
- 快速综述报告生成
- 深度综述报告生成
- 任务元信息和额度流水记录
- 健康检查接口

MVP 阶段后端不做：

- 知网 PDF / CAJ 下载
- 绕过知网登录、验证码、付费或权限控制
- PDF 原文长期保存
- 报告云端历史记录
- 在线文献库
- 团队协作
- 论文代写功能

## 环境变量

本地开发时从 `.env.example` 复制一份 `.env`，并填写真实密钥：

```text
NODE_ENV=development
PORT=3000
HOST=127.0.0.1
DATABASE_URL="file:./dev.db"
DEEPSEEK_API_KEY="你的 DeepSeek API Key"
DEEPSEEK_BASE_URL="https://api.deepseek.com"
DEEPSEEK_MODEL="deepseek-chat"
JWT_SECRET="一串足够长的随机字符串"
```

`DEEPSEEK_API_KEY` 只能保存在后端环境变量中，不能写入浏览器插件或前端构建产物。

## 综述模式

### 快速综述

输入材料：

- 论文题名
- 作者
- 来源
- 发表时间
- 摘要
- 关键词
- 知网详情页链接

输出内容：

- 主题分类
- 代表论文
- 高频观点
- 可能的研究空白
- 建议下载全文精读的论文
- 初步文献综述素材

### 深度综述

输入材料：

- 已采集论文题录
- 摘要和关键词
- 插件端提取的 PDF 全文文本
- PDF 与论文记录的匹配信息

输出内容：

- 单篇论文结构化分析
- 方法与数据对比
- 研究脉络
- 代表论文
- 共同不足
- 研究空白
- 可追溯的文献综述素材

## 数据与隐私原则

- 只保存账号、额度、任务审计所需的最小数据。
- MVP 阶段不长期保存 PDF 全文和报告正文。
- DeepSeek API Key、支付密钥、数据库连接串、JWT 密钥等敏感信息只允许保存在后端环境变量中。
- 不要在日志中记录完整论文全文、完整报告、用户密码或访问令牌。
- 生成后的报告返回给插件端，由用户下载到本地。
